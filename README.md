In the base_structures* folders, there are 2505 database structures collected for classification model validation. The names of these structures are listed in the 'base_names.txt' file. 

The 5param_structures folder contains the XYZ coordinates of the structures obtained from the five-dimensional optimization of the probabilistic parameter and the identification of the vesicle/non-vesicle boundary.
The names of the structures in '5D_all_structures_wl.txt' correspond to the simulation parameters; additionally, each structure is labeled as either a vesicle ('Y') or non-vesicle ('N') structure.

The 2p_* folders contains the XYZ coordinates of the structures obtained from the two-dimensional optimization of the probabilistic parameter and the identification of the vesicle/non-vesicle boundary.
The filenames correspond to the files '2D_*.txt' with modeling energy parameters and an indication of vesicle ('Y') or non-vesicle ('N') structure.
